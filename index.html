Projects

Neuro-Controlled Prosthetic Arm

Engineered a prosthetic arm controlled directly by EEG (brainwave) signals. This project utilized a Random Forest machine learning classifier to achieve 87% grasp detection accuracy. The model was integrated with an Arduino-based embedded system to actuate servo motors in real time, achieving an average response latency of ~250ms from signal to action. The system was validated as a low-cost, portable assistive technology prototype.

Machine Learning EEG Signal Processing Arduino Servo Motors Embedded C/C++

Wearable Device for Respiratory Assessment

Designed a wearable respiratory monitoring device using a 3-pin NTC thermistor to enable temperature-based breath detection. The system implements data acquisition and real-time waveform analysis to calculate breaths per minute (BPM). Over-the-Air (OTA) update capabilities were integrated on the ESP32 to ensure flexible deployment for remote healthcare monitoring.

ESP32 Arduino NTC Thermistor OTA Updates Real-time Data Processing

Over-The-Air Communication with ESP32-S3 Eye

Developed an IoT system using the ESP32-S3 Eye for wireless data transmission and remote monitoring. The project enabled remote device control and real-time data analysis through mobile integration, demonstrating practical expertise in IoT communication protocols and Over-the-Air (OTA) data management.

ESP32-S3 Eye IoT Protocols OTA Updates Mobile Integration
